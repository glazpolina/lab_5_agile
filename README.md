# lab_5_agile
Note App представляет собой веб-приложение для создания заметок. Пользователи могут создавать заметки с метками, менять их статусы, фильтровать их и архивировать. Реализована возможность удаления заметок и ненужных меток. Технологический стек: бэкенд на Python/Flask, фронтенд реализован на HTML+CSS+JavaScript, данные хранятся в JSON-файлах.
## Sprint Review
В спринте были закрыты задачи по созданию базовой структуры веб-страницы (сделан html файл), прописан js файл, и также к заметкам было добавлено поле с метками, которое определяется при создании заметки. Все задачи, минимально необходимые для корректного отображения приложения в браузере, были закрыты. Были не выполнены второстепенные задачи имеющие низкий приоритет связаные с визуальным отображением элементов приложения, не была выполнена задача связанная с добавлением анимаций и адаптивности к разным форматам браузерного окна.
## Sprint Retrospective
Что пошло хорошо: необходимые задачи по фронту были выполнены, поэтому приложение функционально и корректно отображается в браузере.

Что пошло не так: некоторые задачи были слишком обширными, например создание js файла, поэтому эту задачу стоило разбить на несколько подзадач в соответсвии с отдельными элементами приложения, которые необходимо было прописать

В следующем спринте нужно грамотно распределять задачи в соответсвии с их приоритетностью и весом, а также делать задачи спринта меньше, чтобы каждая из них отвечала только за одну функциональную или нефунциональную часть приложения
